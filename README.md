# FocusFlow - Daily Task Planner

A clean and responsive productivity web app built with **HTML, CSS, and JavaScript**.

## Features
- Add and delete tasks
- Mark tasks as completed
- Auto-saves tasks with `localStorage`
- Live progress statistics
- Responsive layout for mobile and desktop

## Run locally
1. Open `index.html` in your browser.

## Push to your GitHub repo
Run these commands from this project folder:

```bash
git init
git add .
git commit -m "Initial commit: FocusFlow task planner"
git branch -M main
git remote add origin https://github.com/hashbuilder60-netizen/web_dev-projects.git
git push -u origin main
```

If Git says a remote already exists, run:

```bash
git remote set-url origin https://github.com/hashbuilder60-netizen/web_dev-projects.git
```
