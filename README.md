## Mixed Tooling Workflow
- Node build: `npm run build:ts`
- Node serve: `npm run serve`
- pnpm: `pnpm run build:ts`
- bun: `bun run build:ts`
- deno: `deno task build`
- Python helper: `python tools/workflow.py build`
- PowerShell helper: `powershell -File tools/workflow.ps1 build`
- Make: `make build`
- Just: `just build`
- Task: `task build`
- cargo-make: `cargo make build`
- uv (Python): `uv run python tools/workflow.py build`
- Poetry: `poetry run python tools/workflow.py build`
- Local preview: `python tools/workflow.py serve 8000`
