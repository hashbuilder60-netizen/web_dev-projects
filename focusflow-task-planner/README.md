# FocusFlow Task Planner

Advanced task management app built with vanilla HTML/CSS/JS.

## Upgrades
- Added quick capture modal (`Ctrl+K`)
- Added List/Board view switching
- Added priority columns in board view
- Preserved advanced filtering, sorting, due dates, and progress analytics

## Inspiration Comparison
- Inspired by workspace task systems and quick-capture workflows seen in Notion-style productivity experiences.

## Run
Open `index.html`.
